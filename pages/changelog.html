Release History

All notable changes to BatchRender are documented here.

v2.1.0
Jan 15, 2025
Latest
GPU Acceleration & Scheduled Jobs
  • New NVENC, QuickSync, AMD VCE hardware acceleration support
  • New Scheduled rendering — set jobs to run at specific time
  • New Folder watch mode — auto-add new files to queue
  • New CSV log export for all render jobs
  • Improved Queue engine rewritten — 40% faster job processing
  • Improved New progress UI with ETA and speed indicators
  • Fixed MKV container metadata not preserved on output
  • Fixed App crash when adding files larger than 50GB
v2.0.5
Dec 1, 2024
Stability & Performance
  • Improved Memory usage reduced by 30% for large queues
  • Improved Better error messages with troubleshooting hints
  • Fixed Audio sync drift for videos over 2 hours
  • Fixed UI freeze when canceling active job
  • Fixed Output filename collision when using same source folder
v2.0.0
Oct 10, 2024
Major Release — BatchRender 2.0
  • New Complete UI redesign — dark mode, modern look
  • New Per-file codec and resolution settings
  • New Auto-retry failed jobs with configurable delay
  • New Desktop notifications on job completion
  • New CLI support for headless/automated workflows
  • Improved Rendering engine — 2.5× faster than v1.x
  • Improved Support for WebM, FLV output formats added
