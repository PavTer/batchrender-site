Installation

BatchRender v2.1.0 · Windows 10/11 (64-bit)

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • 4 GB RAM minimum (8 GB recommended)
  • 500 MB free disk space
  • Optional: NVIDIA GPU (for NVENC), Intel iGPU (for QuickSync), AMD GPU (for VCE)

Installation Steps

  1. Download the installer from your Dashboard → Downloads
  2. Run BatchRender_Setup_2.1.0.exe as Administrator
  3. Follow the installer wizard — choose installation directory
  4. Launch BatchRender from the Start Menu or Desktop shortcut
  5. On first launch, enter your license key (from Dashboard or purchase email)

Uninstallation

Go to Windows Settings → Apps → BatchRender → Uninstall. Your settings and license are preserved and can be restored after reinstallation.

License Activation

How to activate your BatchRender license

Finding Your License Key

Your license key is available in two places: in the confirmation email sent after purchase, and in your Dashboard → My Purchases.

Activation Steps

  1. Open BatchRender
  2. Go to Help → Activate License
  3. Paste your license key and click Activate
  4. The software will verify online and activate within seconds

Transferring to a New Machine

To move your license: on your old machine, go to Help → Deactivate License. Then activate on the new machine using the same key.

If you can't access your old machine, contact [email protected] and we'll reset your activation remotely.

Your First Render

Get started in under 5 minutes

Adding Files

Drag and drop video files onto the BatchRender window, or click Add Files to browse. Supported formats: MP4, MOV, MKV, AVI, WMV, FLV, WebM, and more.

Setting Output Options

Each file shows its output settings in the queue. Click any file to edit: output format, codec, resolution, bitrate, and output folder. Or use Apply to All to set global defaults.

Starting the Queue

Click Start Render. BatchRender will process files one by one (or in parallel if GPU acceleration is enabled). Progress is shown in real time with ETA and speed.

Queue Engine

Managing your render queue

The queue engine is the core of BatchRender. You can add unlimited files, reorder them by drag & drop, and set individual or global output settings per job.

Folder Watch Mode

Enable folder watching in Settings → Queue → Watch Folder. BatchRender will automatically add new files dropped into that folder to the active queue.

Codec Presets

Output formats and encoding settings

BatchRender includes presets for common workflows: H.264 (high compatibility), H.265/HEVC (smaller files), ProRes (editing), DNxHD (editing), VP9 (web), AV1 (web, best quality/size).

Custom Presets

Save your own presets in Settings → Presets → New Preset. Custom presets can be shared as .brpreset files.

GPU Acceleration

Hardware-accelerated encoding

GPU acceleration can provide 4–10× speedup compared to CPU encoding. Supported: NVIDIA NVENC (RTX/GTX series), Intel QuickSync (Intel iGPU), AMD VCE (Radeon).

Enabling GPU Acceleration

Go to Settings → Encoding → GPU Acceleration. BatchRender will automatically detect available hardware encoders. Select your preferred encoder and save.

Troubleshooting

If GPU is not detected: ensure your GPU drivers are up to date. For NVENC, you need driver version 522.25 or newer. For Intel QuickSync, ensure the iGPU is not disabled in BIOS.

Scheduled Jobs

Available in Professional and Studio plans

Scheduled rendering lets you defer a queued job to run at a specific time. This is useful for running overnight renders or during off-peak hours.

Setting a Schedule

Right-click any queue item → Schedule. Choose date and time. The job will start automatically even if the app is minimized, as long as BatchRender is running.

Troubleshooting

Common issues and solutions

App Won't Start

Make sure you're running on Windows 10/11 64-bit. Try running as Administrator. If the issue persists, reinstall the latest version.

License Activation Fails

Check your internet connection. Make sure you haven't exceeded your machine limit. Contact [email protected] for manual activation.

GPU Not Detected

Update your GPU drivers to the latest version. Restart BatchRender after updating. Check that your GPU is not disabled in Device Manager.

Output File is Corrupted

Check that you have enough disk space on your output drive. Try a different output codec. Check the render log for specific error messages.
