How it works

Three steps. Then it just runs.

No timelines, no AI guessing. You define the pipeline. BatchRender applies it to every file.

Step 01

Drop a folder

Drag in 100 .mp4 files, or paste a list of URLs. BatchRender reads everything into a Video Input node — no manual import, no naming conventions.

Step 02

Build a pipeline

Connect nodes: Video Input → Cutter → Add Audio → Add Logo → Output. Set parameters once. The graph is your edit recipe — visible, editable, reusable.

Step 03

Run on the whole batch

One click. Every file goes through the same pipeline. Watch progress live, get a folder of finished MP4s. Save the pipeline as a preset and reuse it next week.

Nodes

Every node earns its place

Six nodes covering the operations creators actually repeat: input, cut, overlay, audio, source, transform.

INPUT

Folder of files in, batch out

Point Video Input at a folder of .mp4 / .mov files. The node reads metadata, durations, sizes — the rest of your pipeline applies to all of them automatically.

EDIT

Cut every file the same way

Frame-accurate Cutter with timecode input. Trim intros, slice into segments, drop the dead air — the same cut rule applied across hundreds of files in one go.

BRAND

Watermarks & logo overlay

Drop a logo or watermark on every video in the batch. Position, scale, opacity — set once, applied consistently. Brand the whole client folder in one render.

AUDIO

Music, voiceover, mute

Add background music, voiceovers, or kill the original audio. Mix multiple tracks. Volume per track. Same audio recipe across every file in the queue.

SOURCE

Pull source from URLs

Built-in URL downloader (yt-dlp). Paste a list of links, BatchRender pulls the source material and feeds it straight into your pipeline — no manual download step.

TRANSFORMSOON

Vertical-blur for social

Convert horizontal video to 9:16 with a blurred background fill. One node, no manual After-Effects work — perfect for Reels, Shorts and TikTok in batch.
